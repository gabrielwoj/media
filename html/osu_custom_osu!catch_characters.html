This section contains customized osu!catch characters. Clicking on an icon will enlarge the image and show a bigger preview.
To download the pack, you can click on the download button located at the top-right corner of the screen (when previewing).
Each pack contains all the four images shown on the preview in a .zip file.
To use these in osu!catch, the contents inside the .zip file should go to the following directory: "osu!/Skins/*YourDesiredSkin*/"
